# RAG Simulator Project

This is a C++ project to simulate a Resource Allocation Graph (RAG) and detect deadlock conditions.

## Features
- Visual text-based graph printing
- Deadlock detection using DFS cycle detection
- Two modes:
  1. Without Deadlock
  2. With Deadlock
- User interactive menu

## Usage
Compile and run the program, then choose:
1. Safe scenario (No deadlock)
2. Deadlock scenario

## Technologies Used
- C++
- GitHub for version control
